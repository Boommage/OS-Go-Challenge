# hfcat

In this directory, you should write a program and compile it into the binary `hfcat`.

After doing so, you can run the tests from this directory by running the `test-hfcat.sh` script. 
If all goes well, you will see:

```sh
prompt> ./test-hfcat.sh
test 1: passed
test 2: passed
test 3: passed
test 4: passed
test 5: passed
test 6: passed
test 7: passed
prompt>
```

The `test-hfcat.sh` script is just a wrapper for the `run-tests.sh` script in
the `tester` directory of this repository.

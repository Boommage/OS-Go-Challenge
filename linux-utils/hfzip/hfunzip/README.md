# hfunzip

In this directory, you should write the program `hfunzip.c` and compile it into the binary `hfunzip` (e.g., `gcc -o hfunzip hfunzip.c -Wall -Werror`)(e.g., `g++ -o hfunzip hfunzip.cpp -Wall -Werror).

After doing so, you can run the tests from this directory by running the `test-wunzip.sh` script. If all goes well, you will see:

```sh
prompt> ./test-wunzip.sh
test 1: passed
test 2: passed
test 3: passed
test 4: passed
test 5: passed
test 6: passed
prompt>
```

The `test-wunzip.sh` script is just a wrapper for the `run-tests.sh` script in the `tester` directory of this repository.
